# DADOS ABERTOS BRASIL

Projeto com intuito de mostrar dados sobre políticos brasileiros.
* 03/2018 -> Informações apenas de deputados.

## Getting Started

```
git clone https://github.com/vfgoncalves/DadosAbertosBrasil.git
```

```
npm i
```

```
ionic serve
```
